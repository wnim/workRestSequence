# Work/Rest Sequence Timer

A visual interval timer for building and running work/rest workout sequences.

## What it does

- **Timeline editor** — drag, resize, and reorder work/rest blocks on a zoomable timeline
- **Playback overlay** — fullscreen countdown with color-coded backgrounds (warm for work, cool for rest) and audio cues
- **Scrubable progress strip** — segmented bar at the bottom shows the full sequence; click or drag to seek
- **Multiple workouts** — save and switch between named workouts
- **Gist sync** — optionally persist workouts to a GitHub Gist

## Running

```
npm install
npm run dev
```

## Keyboard shortcuts

| Key | Action |
|-----|--------|
| `Space` | Play / Pause |
| `Escape` | Stop |
| `← →` | Seek ±5s |
| `Ctrl+S` | Save |
| `?` | Show all shortcuts |
